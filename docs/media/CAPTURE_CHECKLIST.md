# Media capture checklist

The README references the exact filenames below. Drop each file in this folder and the README lights up.

## Prep (once)

- Use a conversation with real, presentable content (no test gibberish). A good demo topic: "explain how neural networks learn", then branch into "what is backpropagation?", "compare with how humans learn", etc.
- Set up: 2-3 branches, one nested branch, one merged branch (violet card in main thread), one compare pair (amber model chips).
- Browser at ~1440px wide, hide bookmarks bar (Cmd+Shift+B), 100% zoom.
- Capture with Cmd+Shift+4 then Space, click the window (clean window shot with no desktop clutter). Turn off the drop shadow if preferred: hold Option while clicking.

## Screenshots (PNG)

| File | What to show |
|---|---|
| `canvas.png` | Canvas view: main thread left, 2-3 branch cards right with connector lines, one active (green). Branch popup open on a message is a bonus. |
| `merge.png` | A branch card with the merge button visible, and the violet "merged from" card in the main thread. Expand the merge card so the synthesis is readable. |
| `compare.png` | Two sibling compare branches side by side, amber model chips visible. Popup with the compare toggle open also works. |
| `graph.png` | Graph view with 4+ nodes: blue main, green active, violet merged with dashed edge. |
| `share.png` | The public share page in map mode (open the link in an incognito window so no app chrome shows). |

## Hero GIF

`hero.gif`, 20-30 seconds, the money shot for Teams:

1. Start on a main thread with 2 messages
2. Click an AI response, type a question in the popup, hit Branch
3. Branch streams in as a card
4. Hit merge, violet card appears in main thread
5. Switch to graph view
6. End on the share page map

Recording options on macOS:
- Kap (free, exports GIF directly): https://getkap.co
- Or Cmd+Shift+5 screen recording (.mov), then convert with Gifski (free, App Store) or ezgif.com

Keep it under ~10 MB for Teams. Tip: record at 1280px wide, 10-15 fps is plenty.

## For Teams specifically

- Teams plays MP4 inline nicely; consider keeping the raw .mov/.mp4 alongside the GIF and posting the MP4 in Teams with a link to the GitHub repo.
- Post format that works well: one-line pitch + hero GIF/MP4 + "try it" link + repo link.
